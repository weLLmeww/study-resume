## About
Very simple resume landing, which demostrate the most important competentions of hypotetical Envelope-developer.


## Functionality
- Interactable header with clickable links nu(number, Telegram, Gmail)
- Sections "Технические навыки", "Обо мне", "Опыт работы", "Образование", "Дополнительная информация"
- Contact form-plug
- Minimalistic stylization with accents for comfort fast-reading
- Freakinly bad sense of humor

## How to open

To open project from your PC you just need to clone git repository
```
git clone https://github.com/weLLmeww/study-resume
cd resume
```
and open index.html
